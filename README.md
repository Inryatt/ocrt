# OCRT

Python script to generate openers for [OpenerCreator](https://github.com/herulume/OpenerCreator)

## About

To get started, run the following (skip `nix develop` if you have nix-direv):

```
$ nix develop
$ poetry run python -m sample_package
Hello, world!
```
